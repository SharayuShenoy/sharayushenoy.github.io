Clustered Data Visualization

Master's Project: R&D Intern at Instana

Project Objective

Instana currently has a dashboard that visualizes their application dependency map. However, this visualization does not adequately represent the vastness of the information and is difficult to navigate as the data increases. The current system uses two layouts to represent this information. The objective of this project is to assess and analyze these two layouts and present their advantages and disadvantages. Further more, this project will also think about improvements based on conducted analysis, and research new solutions and implementations.

Project Activities and Methods

I will undertake the following tasks:

  • Analyze the pros and cons application dependency map based on various factors such as coherence, usability, information architecture etc.
  • Research on existing solutions used to represent similar data and more specifically in a similar domain by conducting an extensive literature review.
  • Recommend modification or alternative solutions based on findings and quote timelines and feasibility to implement the solutions.
  • Implement one solution which has the most impact based on feedback and suggestions from the team.
  • Solution implementation will use the existing tech stack of React.
